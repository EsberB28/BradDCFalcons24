What is "Container"?

Container and Fluid Container are both very important classes in boostrap. The main difference between the two are:

Container

The class "container" provides a fixed width container. This means that, upon using this class, the result would take up a specific portion of the section they're coding. It's width would also change according to the screens viewing it. For instance, it would appear smaller on a phone than it would on a laptop.

Fluid-container

The .container-fluid class provides a full width container, spanning the entire width of the viewport (width is always 100%). This means that, upon using this class, the result would take up the entire view of the section they're coding.
